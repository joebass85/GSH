This is a graphical front-end to the openSSH protocol.
GSH seeks to provide an easy-to-use graphical user interface for openSSH.
GSH must be run from the terminal, inside the gsh/ directory, using the command (without quotes):
"python3 gsh.py".
You must additionally provide a password, if necessary.
This program depends on the subprocess and time modules from the Python Standard Library.
This program also depends on the Tkinter module and the pyautogui module.
Tkinter can be install from your distrobution's repos.
Pyautogui must be installed with the pip3 installer (pip3 install pyautogui)
